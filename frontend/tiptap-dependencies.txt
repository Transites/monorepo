To install Tiptap for this project, run the following command:

npm install @tiptap/vue-3 @tiptap/pm @tiptap/core @tiptap/starter-kit @tiptap/extension-underline @tiptap/extension-link @tiptap/extension-text-align @tiptap/extension-placeholder

This will add the following dependencies to your package.json:

"dependencies": {
  // ... existing dependencies
  "@tiptap/core": "^2.1.12",
  "@tiptap/extension-link": "^2.1.12",
  "@tiptap/extension-placeholder": "^2.1.12",
  "@tiptap/extension-text-align": "^2.1.12",
  "@tiptap/extension-underline": "^2.1.12",
  "@tiptap/pm": "^2.1.12",
  "@tiptap/starter-kit": "^2.1.12",
  "@tiptap/vue-3": "^2.1.12"
}

IMPORTANT: Currently, a fallback editor (FallbackEditor.vue) is being used because the Tiptap dependencies are not installed. After installing the dependencies, update ContentStep.vue to use the TiptapEditor component:

1. Open frontend/src/components/submission/ContentStep.vue
2. Change:
   import FallbackEditor from './shared/FallbackEditor.vue'

   export default {
     name: 'ContentStep',
     components: {
       'rich-text-editor': FallbackEditor
     },

3. To:
   import TiptapEditor from './shared/TiptapEditor.vue'

   export default {
     name: 'ContentStep',
     components: {
       'rich-text-editor': TiptapEditor
     },

After making these changes and installing the dependencies, you can remove the FallbackEditor.vue file if it's no longer needed.
